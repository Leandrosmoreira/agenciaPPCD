# ABISMO CRIATIVO — Identidade da Agência

## Nome
**Abismo Criativo**

## Tagline
*"Da escuridão nasce o conteúdo."*

## Conceito
Agência de publicidade digital especializada em canais dark no YouTube. Produzimos conteúdo de alta qualidade em nichos sombrios, misteriosos e profundos — escatologia, sobrenatural, mistérios históricos, conspirações documentadas, horror bíblico, e temas que exploram os limites do conhecimento humano.

## CEO
**Snayder** — codinome **"O Arquiteto"**
Fundador e diretor criativo. Toma todas as decisões finais de aprovação em cada checkpoint do pipeline de produção.

## Filosofia
- Conteúdo dark com substância — nunca clickbait vazio
- Produção cinematográfica usando IA como ferramenta criativa
- Cada canal é uma divisão independente com identidade própria
- Qualidade sobre quantidade — 1 vídeo excelente por semana por canal

## Ferramentas de Produção
- **Imagens:** Banana 2.0 / Google Imagen
- **Vídeos AI:** Google Veo 3
- **Áudio/Narração:** Suno AI
- **Edição:** CapCut (manual pelo CEO)
- **Publicação:** YouTube Data API v3
- **Pesquisa:** Serper.dev, pytrends, Reddit API, YouTube Data API
- **Orquestração:** Claude Code com subagentes

## Valores da Marca
1. **Profundidade** — Nunca superficial, sempre fundamentado
2. **Mistério** — O tom é sempre enigmático e provocativo
3. **Respeito** — Temas sensíveis tratados com seriedade, não sensacionalismo
4. **Escalabilidade** — Cada processo pode ser replicado para novos canais
5. **Autonomia com supervisão** — Agentes trabalham independentemente, CEO aprova
